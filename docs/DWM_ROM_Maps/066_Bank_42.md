# Bank `0x42`: Text Tables [WIP]
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x108000` - `0x10BFFF`  
**Bank Type:** Swappable  
**Category:** Text  
**Summary:** This bank stores localized text data. It contains three embedded assembly routines[^1] (assumed) for text box control, and a dialogue sector with 113 dialogue sequences.

[^1]: Assembly routines need verification on exact functionality via dynamic debugging.

> **⚠️ Unverified Theory: Text Control Routines**
> The three `0x1D` blocks at `0x40EB`, `0x4108`, and `0x4125` are likely to be assembly routines that handle text box initialization, scrolling, and closing. 
> * The first routine at `0x40EB` appears to perform a bounds check against `0x71` (the exact number of pointers in this bank's Dialogue Pointer Table).
> * **Future Debugging Task:** Set an breakpoint at `0x1080EB`. Step through to see if WRAM `0xC823` holds the active Text ID, and watch if `CALL 05B6` draws the UI.
