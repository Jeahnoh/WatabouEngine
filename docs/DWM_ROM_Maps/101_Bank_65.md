# Bank `0x65`: Garbage Data (Wario Land II)
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x194000` - `0x197FFF`  
**Bank Type:** N/A  
**Category:** Unused / Garbage Data  
**Summary:** Leftover data from *Wario Land II*. Entire bank is similar to Bank `0x65` from WL2.

---

## Memory Map (Granular)
*Break down of the specific blocks within this bank.*

| Offset (Local) | Offset (Global) | Length | Description |
