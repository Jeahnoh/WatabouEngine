# Bank 78: Empty / Garbage Data
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x1E0000` - `0x1E3FFF`  
**Bank Type:** N/A  
**Category:** Empty / Garbage  
**Summary:** Leftover data from Wario Land II. Entire bank matches Bank `0x78` from WL2. Interestingly, also matches a significant portion of the beginning of Bank `0x38` from WL2.

---

## Memory Map (Granular)
*Break down the specific blocks within this bank.*

| Offset (Local) | Offset (Global) | Length | Description |
| :------------- | :-------------- | :----- | :---------- |
| `0x4000` | `0x1E0000` | `0x3da5` | Wario Land II sprite data. Non-standard `2bpp` format. |
| `0x7da5` | `0x1E3DA5` | `0x25b` | 0x00's |
