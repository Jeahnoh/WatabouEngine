# Bank `0x63`: Garbage Data (Wario Land II)
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x18C000` - `0x18FFFF`  
**Bank Type:** N/A  
**Category:** Unused / Garbage Data  
**Summary:** Leftover data from *Wario Land II*. Entire bank matches Bank `0x63` from WL2.
