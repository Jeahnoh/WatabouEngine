# Bank `0x61`: Garbage Data (Wario Land II)
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x184000` - `0x187FFF`  
**Bank Type:** N/A  
**Category:** Unused / Garbage Data  
**Summary:** Leftover data from *Wario Land II*. The first section of this bank closely matches Bank `0x61` from WL2. There appears to be a short sequence of bytes that offset and differ between games, but it is insignificant for DWM. The rest of the bank is empty padding.
