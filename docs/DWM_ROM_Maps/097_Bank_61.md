# Bank 61: Empty / Garbage Data
## Overview
**Local Hex Range:** `0x4000` - `0x7FFF`  
**Global Hex Range:** `0x184000` - `0x187FFF`  
**Bank Type:** N/A  
**Category:** Empty / Garbage  
**Summary:** Leftover data from Wario Land II. Entire bank closely matches Bank `0x61` from WL2. There appears to be a short sequence of bytes that offset and defer between games, but is insignificant for DWM.
